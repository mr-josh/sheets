# Terminal Folders

How to navigate folders in the terminal.

**Get current directory**

```bash
pwd
```

**List files and folders in current directory**

```bash
ls
ls -a # list all files, including hidden ones
ls -l # list extra details
ls -la # combine the 2 above options
```

**Change directory to another folder**

```bash
cd folder-name
cd /absolte/path/to/folder
cd ../relative/path
```
